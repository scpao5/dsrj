# 夺舍日记 —— 三角洲自制同人小游戏

> 作者耗时四到五个月开发的《三角洲行动》题材同人游戏，核心玩法：**夺舍 + 跑酷 + 商店系统**。

## 🎮 游戏简介

这是一款使用 **Scratch** 图形化编程平台开发的独立小游戏，灵感来源于《三角洲行动》中的“夺舍流”玩法。

游戏支持 **手机端** 和 **电脑端**，内置多种音乐和丰富的商店系统，让你在跑酷和夺舍战斗中体验快节奏的博弈乐趣。

## ✨ 游戏特色

- 🦵 **夺舍机制**：还原“卸腿、封烟、补枪、扒装、跑路”的核心玩法
- 🏪 **商店系统**：包含深蓝的盾、威龙喷气、激素枪、甲修等道具，后续还会更新
- 🎵 **多种音乐**：可切换背景音乐，增强游戏节奏感
- 📱 **双端支持**：手机和电脑都能玩
- 🏃 **跑酷环节**：结合动作与策略的玩法
